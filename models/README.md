# 模型目录

存放训练好的模型权重文件(.pth格式)。

## 目录内容

| 文件 | 说明 |
|------|------|
| `mnist_model.pth` | MNIST手写数字识别模型 |
| `smartcar_model.pth` | 智能小车目标识别模型 |

## 模型说明

### MNIST 模型
- 输入: 28x28 灰度图像
- 输出: 10类数字(0-9)
- 架构: CNN (Conv2d -> ReLU -> MaxPool -> FC)

### 智能小车模型
- 输入: 96x96 灰度图像
- 输出: 3类 (交通工具-直行、武器-左、物资-右)
- 架构: 3层CNN + FC
